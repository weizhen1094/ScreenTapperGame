Use left mouse button to click on the buttons

2.When you open the exe, there will be a loading screen

3.Press any key to begin the game

4.Once the game starts, 1-3 buttons will be spawned anywhere on the screen

5.The buttons and points(if any) will be spawned/given every 3 seconds.

9.The highscore is displayed on the center of the screen
